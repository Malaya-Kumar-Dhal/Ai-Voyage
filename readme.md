# Voyage — AI Travel Planner

AI Voyage is an AI-powered travel itinerary generation web application designed to simplify trip planning through intelligent automation. The platform allows users to input their destination, travel duration, budget range, and preferred travel style (such as adventure, relaxation, culture, or food exploration) and instantly receive a personalized, structured, day-by-day travel plan generated using AI.

The primary goal of this project was to combine frontend web development with AI API integration to create a practical, real-world application that demonstrates dynamic content generation, API handling, and responsive UI design.

The application works by collecting user inputs through a clean and intuitive interface, sending a structured prompt request to an AI model via API integration, and dynamically rendering the formatted itinerary response on the page. The system processes AI output and organizes it into a readable, user-friendly format for better usability.
